# Contributing

Help make Nostria better!

## Ways to Contribute

### Use Nostria
- Best contribution: be an active user
- Share feedback
- Report bugs
- Suggest features

### Documentation
- Improve these docs
- Fix typos
- Add examples
- Translate content

### Code
- Fix bugs
- Implement features
- Optimize performance
- Write tests

### Community
- Help other users
- Answer questions
- Share tutorials
- Spread the word

## Getting Started

1. Visit [GitHub](https://github.com/nostria-app/nostria)
2. Read CONTRIBUTING.md
3. Check open issues
4. Fork and submit PR

## Guidelines

- Be respectful
- Follow code style
- Write clear commits
- Test your changes
- Update documentation

## Recognition

Contributors are:
- Listed in project
- Credited in releases
- Part of the community
- Making a difference!
