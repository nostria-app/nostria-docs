# Profile Setup

Your profile is your identity on Nostria. Make it uniquely yours with these customization options.

## Accessing Profile Settings

1. Click your **profile icon** (top right or sidebar)
2. Select **"Edit Profile"** or **"Settings"**
3. Navigate to the **Profile** section

## Essential Profile Elements

### Display Name

Your name as others see it.

**To set:**
1. Find "Display Name" field
2. Enter your name (can be anything)
3. Can include emojis and special characters
4. Save changes

**Tips:**
- Make it memorable and recognizable
- Can be your real name or a pseudonym
- Easy to change anytime

### Profile Picture (Avatar)

Your visual identity.

**To upload:**
1. Click the **avatar area** in Edit Profile
2. Choose **"Upload Image"**
3. Select a photo from your device
4. Crop if desired
5. Save

**Specifications:**
- Format: JPG, PNG, GIF, WebP
- Recommended: Square image (1:1 ratio)
- Size: Max 5MB
- Resolution: 400x400px or higher

**Tips:**
- Use a clear, recognizable image
- Face photos work best for personal accounts
- Logos work for brands/organizations
- Animated GIFs supported

### Bio

Tell people about yourself.

**To write:**
1. Find "Bio" or "About" field
2. Write 2-3 sentences
3. Include interests, profession, or personality
4. Save

**Suggestions:**
- What you do
- Your interests
- Why you're on Nostria
- How people can connect with you

### Banner Image

Large header image for your profile.

**To set:**
1. Click **banner area** in Edit Profile
2. Upload an image
3. Position if needed
4. Save

**Specifications:**
- Format: JPG, PNG, WebP
- Recommended: 1500x500px or 3:1 ratio
- Size: Max 10MB

**Ideas:**
- Landscape photos
- Artwork you created
- Abstract designs
- Brand imagery
- Gradient backgrounds

## Advanced Profile Fields

### Website / Links

Add your website or other links.

**To add:**
1. Find "Website" or "Links" field
2. Enter full URL (include https://)
3. Add multiple links if supported
4. Save

**Common links:**
- Personal website or blog
- Portfolio
- Other social media
- GitHub profile
- Business site

### Lightning Address

For receiving Bitcoin tips.

**To set:**
1. Find "Lightning Address" field
2. Enter your address (e.g., you@getalby.com)
3. Save

**How to get one:**
- Create a wallet with Alby, Wallet of Satoshi, or similar
- They provide a Lightning address
- Copy and paste into Nostria

**Benefits:**
- Receive tips from followers
- Monetize your content
- Support the Bitcoin ecosystem

### NIP-05 Verification

Verify your identity with a DNS-based identifier.

**What it is:**
- Links your Nostr key to a domain
- Shows as verified badge on your profile
- Format: username@domain.com

**To set:**
1. Get a NIP-05 identifier from:
   - Your own domain (self-hosted)
   - A verification service
   - Some Lightning wallets include it
2. Enter it in "NIP-05" field
3. Save
4. Wait for verification (usually instant)

**Popular services:**
- nostr.directory
- getalby.com
- Your own domain (if technical)

**Benefits:**
- ‚úÖ Verified badge
- ‚úÖ Easier to find
- ‚úÖ Professional appearance
- ‚úÖ Prevents impersonation

## Profile Privacy Settings

### Who Can See Your Profile

Usually profiles are public on Nostr, but you can:
- Block specific users
- Filter who can tag you
- Control DM permissions

### Content Warnings

Flag your profile if you post:
- NSFW content
- Controversial topics
- Sensitive material

Set this in Privacy settings.

## Common Profile Questions

**Q: Can I change my display name?**
A: Yes, anytime! It's not tied to your identity.

**Q: Why doesn't my profile picture show?**
A: Check image format, size, and internet connection. Try uploading again.

**Q: How do I get verified?**
A: Set up a NIP-05 identifier linked to a domain you control.

**Q: Can I have multiple Lightning addresses?**
A: Most clients support one, but you can change it anytime.

**Q: Is my profile private?**
A: Nostr profiles are generally public, visible to anyone.
