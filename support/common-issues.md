# Common Issues

Quick solutions to frequently encountered problems.

## Login Issues

**Problem**: Can't log in
**Solution**: Verify private key, clear cache, try different browser

**Problem**: Wrong account loads
**Solution**: Logout, clear cookies, login with correct key

## Posting Issues

**Problem**: Posts not appearing
**Solution**: Check relay connection, refresh, wait 30 seconds

**Problem**: Can't upload images
**Solution**: Check file size (<5MB), format (JPG/PNG), internet connection

## Performance

**Problem**: Slow loading
**Solution**: Reduce relay count, clear cache, close other tabs

**Problem**: High CPU usage
**Solution**: Disable hardware acceleration, update browser, reduce active tabs

## Notifications

**Problem**: Not receiving notifications
**Solution**: Check browser permissions, enable in settings

## General

**Problem**: Something's broken
**Solution**: Refresh page, clear cache, update browser, restart app
