## Reporting Bugs

Found a bug? Help us fix it:

1. Check if it's already reported
2. Go to [GitHub Issues](https://github.com/nostria-app/nostria/issues)
3. Create new issue with:
   - Clear description
   - Steps to reproduce
   - Expected vs actual behavior
   - Screenshots if applicable
   - Browser/OS version

## Feature Requests

Have an idea?
- Post on Nostria with #NostriaFeature
- Open GitHub issue
- Discuss with community
- Vote on existing requests
