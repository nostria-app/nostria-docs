# Content Backup & Restore

Protect your data with automatic backups (Premium feature).

## Manual Backup

**Free for everyone:**
1. Go to Settings → Data
2. Click "Export Data"
3. Download your data
4. Store securely

**Includes:**
- All your posts
- Profile information
- Follow lists
- Settings

## Premium Automatic Backup

With Nostria Premium:
- Automatic cloud backups
- Daily snapshots
- Version history
- One-click restore

[Learn about Premium →](premium-features.md)

## Restoring Data

1. Settings → Data
2. Click "Import Data"
3. Select backup file
4. Confirm import
5. Data restored!

[Learn about data management →](import-export.md)
