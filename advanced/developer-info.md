# Developer Information

Build on Nostr and integrate with Nostria.

## For Developers

### Nostr Protocol
- Open protocol
- Well-documented
- Active community
- Easy to implement

### Resources
- [Nostr Protocol Specs](https://github.com/nostr-protocol/nostr)
- [NIPs Repository](https://github.com/nostr-protocol/nips)
- [Nostr Tools](https://github.com/nbd-wtf/nostr-tools)
- [Implementation Guide](https://nostr.how)

### Building Nostr Apps
1. Understand the protocol
2. Choose NIPs to support
3. Implement relay communication
4. Handle key management
5. Build your UI

### Nostria Integration
- Use same keys across all Nostr apps
- Your app works with Nostria users
- Interoperable by design

## API Access

Nostria uses standard Nostr protocol:
- WebSocket connections
- JSON events
- No proprietary API needed
- Direct relay access
