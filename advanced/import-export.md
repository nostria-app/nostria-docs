# Import & Export Data

Move your data between apps and create backups.

## Export Your Data

1. Settings → Data Management
2. Choose what to export:
   - Profile data
   - Posts
   - Follow lists
   - Settings
3. Click "Export"
4. Save file securely

## Import Data

1. Settings → Data Management
2. Click "Import"
3. Select file
4. Choose what to import
5. Confirm

## Data Formats

- JSON format
- Compatible with other Nostr clients
- Human-readable
- Easy to parse

## Use Cases

- Backup before major changes
- Switch between Nostr clients
- Migrate to new device
- Share follow lists
- Archive your history
