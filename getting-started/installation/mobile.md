## Using the Web App on Mobile Now

### Add to Home Screen

**On iOS (Safari)**:
1. Visit [nostria.app](https://nostria.app)
2. Tap the Share button (square with arrow)
3. Scroll and tap "Add to Home Screen"
4. Name it "Nostria" and tap "Add"

## Mobile-Specific Tips

### Battery Saving
- Disable background refresh when not needed
- Use dark mode to save battery (OLED screens)
- Limit notification frequency

### Data Usage
- Download media only on WiFi
- Enable data saver mode
- Cache content for offline viewing

### Security
- Enable biometric lock on your device
- Don't save keys in browser autofill
- Use a password manager for keys
- Enable device encryption
